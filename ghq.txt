== NAME

ghq - Manage remote repository clones

This is a subspecies of ghq and the unofficial repository.

== DESCRIPTION

show [super repository](https://github.com/motemen/ghq).

== ADDITIONS

Category system.

    We can add a category for the repository which will be fetched by ghq get.
    We can also show the list of the repositories which are specified a category.

== [[installing]]INSTALLATION

----
go get github.com/jmatsu/ghq
----

Or clone the https://github.com/jmatsu/ghq[repository] and run:

----
make install
----
